  - Solution 1, convert to string, faster, because the slicing operation is optimized in C
```python
class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        x = str(x)
        return x == x[::-1]
```

```python
class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        y = 0
        x_copy = x
        while x>0:
            y = y*10+(x%10)
            x = int(x/10)
        return x_copy == y
```
